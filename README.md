## 功能说明
将下载好的图片（本子）转换为PDF
## 使用说明
1. 安装第三方库：
  ```shell
  pip install jmcomic -i https://pypi.org/project --upgrade
  pip install pillow
  pip install pyyaml 
  ```
2. 可直接运行main.py进行下载和转换，但需要预先配置config路径,
```shell
manhua = ['146417']  
for id in manhua:
  jmcomic.download_album(id,loadConfig)
```
可取消上述代码注释，manhua = ['146417']  中可填写多个需要下载的本子，格式:['12314545','2321415']以此类推；
